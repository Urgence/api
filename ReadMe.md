## Urgence  API  


## Installation

```bash
$ yarn
$ cp .env.sample .env
```

## Running the app

```bash
# launch docker for start bdd
$  docker-compose -f .docker/docker-compose.yml  up
# for stop docker
$  docker-compose -f .docker/docker-compose.yml  down

# development
$ yarn dev

# start
$ yarn build
$ yarn start

# test
$ yarn test
```
